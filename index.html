Front-end Tutorial

Välkommen till vår tutorial. På denna hemsida kommer du lära dig de grundläggande kunskaperna som krävs för att bygga din egen webbplats. Här går vi igenom hur man gör en webbplats inom kategorierna HTML, CSS, Javascript och Photoshop. Alla exempel vi visar är baserade på hur vi gjort på den här webbplatsen steg för steg.

Vi rekommenderar att man följer kategorierna steg för steg om man vill göra sin egen webbplats utan förkunskaper. Annars har vi ett index för våra exempel om man vill veta mer kring någonting specifikt.

På HTML sidan går vi igenom hur man man strukturerar upp en webbsidas innehåll. CSS sidan går igenom hur man stylar HTML sidan. Javascript går igenom hur man skapar en sökfunktion m.m. Photoshop sidan går kort igenom hur man anpassar bilder för webben.
