HTML

Intro

Varje hemsida är uppbyggd med någon form av html kod som bas, utan detta får man ingen struktur på hemsidan och dess innehåll. Här kommer vi gå igenom hur vi byggt upp webbsidan du är inne på och vad varje del av koden gör.

HEADER

Vi börjar med head elementet. Detta element innehåller information om dokumentet t.ex. titeln på webbsidan, vilket språk sidan är skriven på samt länkar till eventuella CSS och Javascript filer som används. Såhär ser vår grundstruktur ut för head taggen.

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Frontend Tutorial - HTML</title>
    <link href="https://fonts.googleapis.com/css?family=KoHo|Open+Sans|Ubuntu+Condensed" rel="stylesheet">
    <link rel="stylesheet" href="style.css">
</head>

De flesta webbsidorna du besöker använder liknande taggar och brukar ofta vara med i färdiga html mallar. Dessa taggar ändrar gör inte mycket visuellt för webbplatsen utan fungerar som sagt mer som information om sidan.

Semantic Elements

Body elementet kommer innehålla majoriteten utav sidans innehåll men det kommer vara uppdelat i flera mindre sektioner. Dessa sektioner kallas för “Semantic elements” och de är ofta väldigt tydliga med vad de ska innehålla. Några exempel på dessa element är header, nav, article och footer. Med hjälp utav dessa kommer vi få grundstrukturen i vår html kod.

<body>
    <header>
        <nav>
        </nav>
    </header>
    <main>
        <article>
        </article>
    </main>
    <footer>
        <address>
        </address>
    </footer>
</body>

Dessa element kommer inte synas direkt på webbsidan förrän du faktiskt börjar lägga till riktigt innehåll. De är viktiga för att sidan ska få sin struktur men också för sökmotorer då datorn lättare kan förstå innehållet på webbsidan. Detta blir väldigt viktigt när man börjar fokusera på SEO (Search Engine Optimization).

Innehåll

När vi väl har detta på plats kommer vi börja fylla dessa taggar med vårt innehåll. Detta kommer vi göra med hjälp utav mindre taggar t.ex. h1-h5 för titlar, p för paragrafer och li för listor etc.

<h2>Innehåll</h2>
    <p>När vi väl har detta på plats kommer vi börja fylla dessa taggar med vårt innehåll. 
    Detta kommer vi göra med hjälp utav mindre taggar t.ex. h1-h5 för titlar, 
    p för paragrafer och li för listor etc.</p>
