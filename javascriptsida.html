Javascript

Kort sagt är Javascript ett programmeringsspråk som tolkas av en webbläsare. Det används för att skapa funktioner till webbplatser och på så vis göra de mer funktionella. Vanliga funktioner kan vara att manipulera bilder, godkännande av form-listor eller dynamiska ändringar av sidans innehåll.
I den här tutorialen kommer du lära dig hur man tillämpar Javascript till din Html och CSS och ge dig en förståelse till vad du kan använda Javascript för.

Koppla Javascript till din Html.

Till att börja med behöver du inkludera en script-tagg till din Html. Du kan antingen välja att göra en script tagg längst ner inuti htmls body-tagg och skriva din kod där. Man kan också skriva koden i en separat fil och länka till den utifrån script-taggen, väljer du att länka kan du inte skriva någon kod i taggen.
Eftersom koden blir lättare att läsa och hålla reda på med en separat fil väljer vi att göra det.
Skapa en ny fil med din text-editor och välj Javascript som format. Det är smart att välja relevanta namn. Vi döper vår till script.js 


                        //tagg utan länk
                        <script</script>

                        //script tag med länk till separat JS fil. 
                        <script src=”script.js”></script>

Variabler

Variabler i programmering fungerar som en behållare för ett värde. Det finns fem olika datatyper i Javascript, Boolean(true or false), null, undefined, number eller string. javascript kommer tolka din variabel utifrån en av dessa beroende på vad du ger den för värde. Ger du den en siffra så tolkas den som number, innehåller den en text tolkas den som string. Du deklerar en variabel genom att skriva let ettNamn = “värde”;. 


                        //ger x värdet 5.
                        let x = 5;

                        //ger y värdet 10.
                        let y = 10;

                        //ger z värdet x + y, alltså 15.
                        let z = x + y;

DOM och hur man manipulerar den.

Med Htmls DOM, kan Javascript få tillgång till och manipulera alla element från HTML-trädet. När en webbsida öppnas så skapar webbläsaren ett Document Object Model av sidan. DOM skapar då objekt av alla element som du kan manipulera med JS genom att anropa dem i din kod. let rubrik = document.getElementByID(“domExample”); Tex, hade huvudrubriken på denna sida haft id=domExample. Hade vi hämtat rubriken.

Egenskaper

Vill vi bara hämta texten elementet innehåller. Hade vi kunnat använda oss av en egenskap i DOM som heter innerHTML. Variabeln hade då skrivits: let rubrik = document.getElementByID(“domExample”).innerHTML;
Snutt på ett exempel som visar hur man tar rubrikens innehåll och lägger det värdet i en paragraf.

Funktioner

En javascript funktion är en kodsnutt skapad för att utföra en specifik uppgift. Funktioner exekveras när någonting anropar dem. *Exempel kod som visar hur man deklarerar en funktion och ett exempel.*

Events

I denna tutorial exekverar vi våra funktionerna genom javascript events. Vi använder oss av tre olika versioner av dessa: onmouseenter(exekverar funktionen genom att ha hovra över ett element ), onmouseleave(triggar funktionen när man slutar hovra över ett element) som hör ihop och onkeyup (exekverar funktionen när man har markerat ett inputfält) *Exempel, onmouseenter, onmouseleave*

arrayer

En array används för att spara flera värden i samma variabel. De fungerar ungefär som en lista, *se kodsnutten för exempel*. Arrayer är väldigt användbara om du behöver spara många värden och enkelt vill hämta dem. Arrayer kan skrivas på många olika sätt beroende på dess syfte, de har också många användbara metoder. Vi behöver inte gå in detta så djupt då vi enbart använt dessa inom ett användningsområde. *kodsnutt gå igenom en array som innehåller ett par variabler, skriv ut ett av arrayens index*

Loopar

Loopar används för att repetera ett kodblock fram tills att ett visst tillstånd blir mött. Tillstånden brukar även kallas statements och för detta kan man använda tex en if-sats. Det finns olika former av loopar som tex, while, do-while och for-loopen. Vi går igenom for-loopen nu då vi enbart använder oss av den under vår tutorial. *kodsnutt på en for-loop som loopar igenom en array och bryts när ett visst index möts.*
