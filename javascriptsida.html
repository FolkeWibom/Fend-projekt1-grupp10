Javascript

Kort sagt är Javascript ett programmeringsspråk som tolkas av en webbläsare. Det används för att skapa funktioner till webbplatser och på så vis göra de mer funktionella. Vanliga funktioner kan vara att manipulera bilder, godkännande av form-listor eller dynamiska ändringar av sidans innehåll.
I den här tutorialen kommer du lära dig hur man tillämpar Javascript till din Html och CSS och ge dig en förståelse till vad du kan använda Javascript för.

Koppla Javascript till din Html.

Till att börja med behöver du inkludera en script-tagg till din Html. Du kan antingen välja att göra en script tagg längst ner inuti htmls body-tagg och skriva din kod där. Man kan också skriva koden i en separat fil och länka till den utifrån script-taggen, väljer du att länka kan du inte skriva någon kod i taggen.
Eftersom koden blir lättare att läsa och hålla reda på med en separat fil väljer vi att göra det.
Skapa en ny fil med din text-editor och välj Javascript som format. Det är smart att välja relevanta namn. Vi döper vår till script.js 


//tagg utan länk
<script> </script>

//script tag med länk till separat JS fil. 
<script src=”script.js”></script>

Variabler

Variabler i programmering fungerar som en behållare för ett värde. Det finns fem olika datatyper i Javascript, Boolean(true or false), null, undefined, number eller string. javascript kommer tolka din variabel utifrån en av dessa beroende på vad du ger den för värde. Ger du den en siffra så tolkas den som number, innehåller den en text tolkas den som string. Du deklerar en variabel genom att skriva let ettNamn = “värde”;. 


//ger x värdet 5.
let x = 5;

//ger y värdet 10.
let y = 10;

//ger z värdet x + y, alltså 15.
let z = x + y;

DOM och hur man manipulerar den.

Med Htmls DOM, kan Javascript få tillgång till och manipulera alla element från HTML-trädet. När en webbsida öppnas så skapar webbläsaren ett Document Object Model av sidan. DOM skapar då objekt av alla element som du kan manipulera med JS genom att anropa dem i din kod. 


//Hade hade huvudrubriken på denna sida haft id=domExample. Hade vi hämtat elementet såhär:
let rubrik = document.getElementByID(“domExample”);

Egenskaper

Dessa objekt har olika egenskaper och attributet du kan hämta genom att lägga till olika metoder. Vill vi bara hämta texten elementet innehåller. Hade vi kunnat använda oss av en egenskap i DOM som heter innerHTML. 


//Vill vi enbart hämta texten, hade vi deklarerat variabeln såhär:
let rubrik = document.getElementByID(“domExample”).innerHTML;

//vill vi ge domExamples värde till en annan rubrik med id=domCopy gör vi såhär:
document.getElementById('domCopy').innerHTMl = rubrik;

Vi behöver inte gå in detta så djupt då vi enbart använt dessa inom ett användningsområde. 


//funktion som innehåller tre variabler med olika värden
function myFunction(){

let y = 15;
let z = 20;

let numberArr = [x, y, z]

index startar alltid från 0. Här under hämtar vi variabeln x.

Vi går igenom for-loopen nu då vi enbart använder oss av den under vår tutorial. *kodsnutt på en for-loop som loopar igenom en array och bryts när ett visst index möts.*

3- i++; här plussar den på loopens index ifall loopen inte bryts på ett index och fortsätter.

for(let i = 0; i < numberArr.length; i++){

Navigationsbar

steg1 - funktionerna

steg1. funktion dropDownfuncHtml(). Denna funktion anropar html elementet med id mydropdownHTML som vi sedan lagt en metod .classlist.toggle(“show”) som lägger till klassen “show”. Show är en CSS klass vi skapat som gör att listan syns. Utan denna klass göms html Elementet.

För att bara en av dropdown listorna ska kunna vara öppen åt gången. Har vi deklarerat ut tre variabler som innehåller de andra listorna. Vi har lagt dessa inuti en array som vi loopar igenom. Den kollar då ifall någon av de andra listorna är öppen och koden inuti if-satsen stänger då de andra eventuella öppna listorna. 



// Öppnar Dropdown listorna och stänger andra öppnade
function dropDownFuncHtml() {
document.getElementById("myDropdownHTML").classList.toggle("show");
                            
let dropCss = document.getElementById("myDropdownCSS");
let dropJs = document.getElementById("myDropdownJS");
let dropPs = document.getElementById("myDropdownPS");
let dropArr = [dropCss, dropJs, dropPs];
for (let i = 0; i < dropArr.length; i++) {
    let closeDrop = dropArr[i];
    if (closeDrop.classList.toggle("show")) {
        closeDrop.classList.remove('show');
        }
    }
}

Denna funktion är i princip samma sak som den undre delen av den tidigare funktionen. Den använder vi för att stänga alla öppna listor. 



function closeDropDown() {
let dropdowns = document.getElementsByClassName("dropdown-content");
                              
for (let i = 0; i < dropdowns.length; i++) {
let openDropdown = dropdowns[i];
if (openDropdown.classList.contains('show')) {
openDropdown.classList.remove('show');
        }
    }
}

Sökfält

steg 1 - funktionen

Börjar med att deklarara ut en variabel som innehåller sökfältets värde. sedan har vi gjort en array som innehåller idt på alla länkar sidan innehåller. Vi loopar igenom arrayen och kollar ifall inputsearch variabeln matchar någon av länkarnas id. Om värdet finns kommer a-taggen inuti sökknappens href attributet ändras till samma som länkens. 


/*Funktion som triggas när man är i inputfältet med id inputSearch*/
function searchFunction() {

    /*Sparar texten man skrivit i inputfältet i en variabel*/
let inputSearch = document.getElementById('inputSearch').value;

/* Array med med a-taggarna i listan*/
let arr = [document.getElementById('start'), document.getElementById('html'), document.getElementById('css'),
document.getElementById('javascript'), document.getElementById('photoshop'), document.getElementById('script-taggen'),
document.getElementById('variabler'), document.getElementById('dom'), document.getElementById('funktioner'),
document.getElementById('loopar'), document.getElementById('navigationsbar'), document.getElementById('sökfält'),
document.getElementById('arrayer')];

/*Loopar igenom arrayen för att se om inputSearch variabeln matchar något index i arrayen */
for (let i = 0; i < arr.length; i++) {
                            
/*Om de matchar ska länkens href attributet kopieras över till sökknappens href attribut*/
    if (inputSearch == arr[i].id) {
        let link = arr[i];
        document.getElementById('searchBtn').href = link;
        }
    }
}
